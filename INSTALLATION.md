# 📥 How to Install NexLinks

This guide will help you install NexLinks on your device. Choose your platform below.

---

## 🤖 Android (APK)

1. **Download** the `app-release.apk` file from the [Downloads page](../README.md#downloads).
2. Tap on the downloaded file.
3. If you see a warning saying **"Install from unknown sources"**, tap **Settings** and turn on **"Allow from this source"**.
4. Tap **Install**.
5. Once finished, tap **Open** to start the app!

---

## 🪟 Windows

1. **Download** the `NexLinks_Installer.msix` file.
2. Double-click the file.
3. You will see a window asking to install **NexLinks**.
4. Click **Install**.
   - *Note: If a "SmartScreen" warning appears, click "More Info" -> "Run Anyway".*
5. The app will install and automatically launch. You can find it later in your **Start Menu**.

---

## 🐧 Linux

1. **Download** the `linux_app.tar.gz` file.
2. Right-click the file and select **Extract Here**.
3. Open the extracted folder.
4. Double-click the **nexlinks** file to run it.
   - *Pro Tip: You can verify the file is executable by Right-Click -> Properties -> Permissions -> check "Allow executing file as program".*

---

## 🍎 iOS (iPhone/iPad)

*> Note: Installing on iOS currently requires a developer setup or TestFlight.*

1. **Download** the `ios-ipa` file.
2. You can install this using tools like **AltStore** or **Sideloadly**.
3. **AltStore Method**:
   - Install AltStore on your phone.
   - Open AltStore -> My Apps -> Tap the **+** button -> Select the `NexLinks.ipa` file.

---

## 🌐 Web

No installation needed! Just visit the hosted link (if provided) or open the `web` folder content in a browser.
